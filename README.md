this project is about  TIM+ADC+DMA 

ADC sample time=53.33khz

ADC1  PB14 IN5   

ADC2  PB2  IN12

ADC3  PB13 IN5

ADC4  PB15 IN5

USART1_TX PA9   USART_RX PA10

TIM20_PSC=24   TIM_ARR=9  TIMCLOCK=80MHZ

KEY1 PC13

LED2 PA5


this project start with keil5  528

support st—link v3
